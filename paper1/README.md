## Overview

This repository contains all code to reproduce the analyses, figures, and documents
in the paper. The model is a fused optimal-control (OC) + individual-based model
(ABM) that asks what grandmothering does to (a) the age at menopause and (b) the
post-reproductive lifespan (PRLS) separately. A key biological innovation in the
revised model is that both depend on realistic age-graded foraging productivity
(Kaplan et al. 2000) and effort-dependent somatic damage accumulation consistent
with Disposable Soma Theory (Kirkwood 1977; Kirkwood & Rose 1991).

## Key model parameters

All parameters live in `model.Params`. Defaults reproduce the calibrated model:

| Parameter | Default | Description |
|-----------|---------|-------------|
| `B` | 1.0 | Maternal grandmothering strength (≈ 2–3 extra surviving grandoffspring) |
| `B_pgm` | 0.0 | Paternal grandmother strength (0 = off; use 1.0 for two-gm scenario) |
| `p_u` | 0.05 | Paternity uncertainty (reduces PGM relatedness; relevant when `B_pgm > 0`) |
| `use_prod_curve` | True | Apply age-graded productivity g(a) (Kaplan et al. 2000) |
| `delta_base` | 0.012 | Background (effort-independent) somatic damage rate |
| `delta_rep` | 0.006 | Effort-dependent damage increment (Kirkwood DST): δ(u) = δ₀ + δ_rep·u² |
| `gm_a_zero` | 90 | Age at which grandmother helping efficacy reaches zero |
| `mm_scale` | 1.0 | Set to 0.0 to disable maternal-mortality costs (for stepwise comparison) |
| `rate` | 0.0 | Population growth rate r; 0 = R₀ stationary currency |
| `a_ceiling` | 50 | Atresia ceiling (age of menopause in exogenous-Tr runs) |

### Two-grandmother scenario

The OC model tracks both grandmothers explicitly:
```python
from model import Params, run_cell
p = Params(B=1.0, B_pgm=1.0, p_u=0.05)   # MGM + PGM, 5% paternity uncertainty
sol, sim = run_cell(p, free_u=False)
print(sim['Td'], sim['PRLS'])   # → 63.0, 13.0  (matches Gurven-Kaplan target)
```

The ABM does not track paternal lineages. To approximate the two-grandmother
scenario in the ABM, pass the effective combined B:
```python
B_eff = 1.0 + 1.0 * (1 - 0.05)   # = 1.95
from abm import run_abm
H, s = run_abm(B=B_eff, ...)
```

## Calibration targets

The model is calibrated to Gurven & Kaplan (2007) forager-horticulturalist
demography:

| Condition | l₇₀ (target: 0.40–0.47) | Modal death (target: 68–78 yr) |
|-----------|------------------------|-------------------------------|
| B=0 (no grandmothering) | 0.18 | ~50 |
| B=1 (single MGM) | 0.30 | ~73 |
| B=1 MGM + B=1 PGM (p_u=0.05) | **0.43** ✓ | **~75** ✓ |
| B=3 (single, strong) | 0.43 | ~75 |

Note: with the revised model (age-graded productivity + Kirkwood damage), a
*single* grandmother at the empirical B=1 no longer matches the life-table target.
*Both* grandmothers together are required, consistent with empirical meta-analyses
showing positive effects of both MGM and PGM (Sear & Mace 2008).

## Key biological innovations relative to prior formal treatments

1. **Age-graded productivity g(a)** (Kaplan et al. 2000): net foraging production
   rises to a peak at ~38 yr and declines thereafter, scaling both reproductive
   yield and grandmother help capacity. Implemented via `productivity_curve()` in
   `model.py`, applied in both OC and ABM.

2. **Effort-dependent somatic damage δ(u)** (Kirkwood 1977; Kirkwood & Rose 1991):
   `δ(u) = δ_base + δ_rep · u²`, where reproductive effort `u` drives additional
   ROS production and damage (Speakman 2008; Costantini et al. 2010). Implemented
   in `solve_dp()`, `simulate_policy()`, `abm.py`, and `invasion.py`.

3. **Two-grandmother extension**: maternal grandmother (r = 0.25, certain) and
   paternal grandmother (r = 0.25 × (1 − p_u), discounted by paternity uncertainty)
   are modelled additively in the OC. The ABM approximates this via a combined B_eff.

4. **Ne-based effective-selection thresholds** on invasion figure (Figure 6):
   dashed lines at s = 1/(2Ne) for Ne = 30 (hunter-gatherer band; Hill et al. 2011)
   and Ne = 500 (dialect tribe; Birdsell 1973) show where selection overcomes drift.

### ABM / OC consistency note

The ABM tracks only the *maternal* grandmother lineage. The OC models both
grandmothers explicitly via `B_pgm` and `p_u`. To approximate the OC's
two-grandmother result in the ABM, set `B = B_eff ≈ 1.9` (= 1.0 + 1.0 × 0.95)
as described in the **Two-grandmother note** above.

The invasion experiment (`invasion.py`) holds the maintenance schedule fixed when
varying cessation age. The OC finds a global fitness maximum at Tr ≈ 40 because
it simultaneously co-optimises maintenance: a female who stops early invests more
in repair and lives longer to help. The invasion at fixed maintenance shows s ≈ 0
near the ceiling, consistent with the OC's finding of a nearly flat fitness landscape
over Tr (< 1% fitness gain at B = 1). This apparent discrepancy is not a
contradiction — it is a statement about the co-evolutionary nature of the optimum.
